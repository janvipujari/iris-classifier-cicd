# Iris Classifier CI/CD Project

This project demonstrates an end-to-end Machine Learning pipeline with CI/CD integration using GitHub Actions.

## Features
- Iris classification using Random Forest
- Flask API for predictions
- Unit testing with pytest
- Automated CI/CD pipeline

## Tech Stack
- Python
- Flask
- Scikit-learn
- GitHub Actions

## Run Locally
```bash
python train_model.py
python app.py
